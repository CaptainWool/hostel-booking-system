# Hostel Booking System

## Project Overview
The Hostel Booking System is a web application designed to facilitate the management of hostel bookings for travelers and hostel owners. It provides an easy-to-use interface for booking, managing reservations, and processing payments.

## Features
- **User Registration**: Allow users to register and create an account.
- **Search Functionality**: Users can search for hostels based on various filters such as location, price range, and availability.
- **Booking Management**: Users can easily book available rooms, view their bookings, and modify or cancel them.
- **Admin Panel**: Hostel administrators can manage listings, bookings, and user accounts.
- **Payment Integration**: Secure payment processing for user transactions.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Payment Processing**: Stripe API

## Installation Guide
1. **Clone the repository**:
   ```bash
   git clone https://github.com/CaptainWool/hostel-booking-system.git
   cd hostel-booking-system
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the application**:
   ```bash
   npm start
   ```

## Usage
- To use the application, go to `http://localhost:3000` in your web browser.
